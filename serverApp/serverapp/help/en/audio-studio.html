Audio Studio

Console for generating artwork audio and audio guide introductions.

1. Generate voice audio

The audio text is taken from the content and the voice must be selected from the available voices in the language ( voice library ).

2. Add music intro

This step is optional. Music can be selected from the files available in the system ( music library ) or paste a URL to the music file so the system can download and use it. In this case the user is responsible for having the usage license for that music file.

3. Integrate

If the final audio is integrated into the content, it will replace the existing content audio -if any-, otherwise it will remain saved in the content's audio studio but will not be used in the audio guide.
