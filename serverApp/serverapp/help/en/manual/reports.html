There are access reports for the Visitor Guide in the following entities: Venue, Audio guide, Artwork audio, Artwork.

Venue

Venue users

Counts the number of user sessions; the system does not record the identity of the people who access the content, only the device. If a user accesses from more than one device the system will count it as two since it has no way of knowing it is the same person.

Languages

Sessions by language may add up to more than the total users because a user may change language during the session.

Sessions per audio guide

Sessions per audio guide count the number of unique users who accessed each audio guide. If the sum is greater than the total users it is because users can access several audio guides in each session.

Audio guide

Audio guide users

Counts the number of user sessions that accessed the audio guide and the artwork audios.

Languages

Sessions by language may add up to more than the total users because a user may change language during the session.

Artwork audio

Sessions per artwork audio correspond to the audio guide; the artwork may have visits from other audio guides (for example the accessible audio guide of the same exhibition).

Artwork audio

Artwork audio users

Counts the number of user sessions that accessed the artwork's audio in all languages.

Languages

Sessions by language may add up to more than the total users because a user may change language during the session.

Other guides for the same artwork

If the artwork is included in another audio guide, this link allows you to view those visits.

Artwork

To see the total visits to an artwork across all audio guides it is included in, go to the reports section of the artwork's page.
