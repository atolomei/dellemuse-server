Creating exhibitions

Exhibitions are created on the venue's exhibitions page. Each exhibition has artworks and objects that are added from the venue's archive, and can be permanent or temporary. Temporary exhibitions have a start date and an end date in the Venue's time zone.

That means that if an exhibition has a start date of May 1st at the National Museum of Fine Arts of Argentina, it will be active in the system from midnight Argentina time on May 1st.

The main information form is in the venue's primary language; and there are also information forms in each of the venue's secondary languages. The 'order' field is used to sort exhibitions in the visitor application (in case the venue's configuration is set to order and not alphabetical).

Automatic translation to each secondary language of the venue is done using the primary language text as the source.

Exhibitions can have the status Editing, Published or Deleted (recycle bin). Only exhibitions with Published status are shown in the visitor guide. During content editing and audio guide generation the exhibition must be in Editing status; once it is ready for visitors it must be changed to Published status so that it appears in the Visitor Guide.

Artworks and objects in the exhibition

Set of artworks and objects in the exhibition. They are added from the venue's artworks and objects archive. If an item does not appear, it must be added to the venue's artworks and objects archive before it can be added to the exhibition.

QR code and audio number of the exhibition

The system generates a QR code and an audio number for the exhibition, which must be placed in the exhibition room — normally next to the curatorial text that is printed or mounted on the wall. The visitor scans that QR and can listen to the introductory text in their language. They can also use the search on their phone and search by the exhibition's audio number.

Exhibition audio guides

Each exhibition can have one or more Audio guides, for example the general audio guide and an Accessible audio guide for people with reduced vision (see the Audio guide manual)
