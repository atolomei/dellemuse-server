Each Institution and its Venues can manage their users independently.

For this purpose there are user roles specific to the institution and each venue.

A Role is a set of permissions, and a user has the permissions granted by all their assigned roles. The roles are: institution admin, venue admin and venue editor.

Roles of the institution Museo Nacional de Bellas Artes and its (sole) venue
Venue editor

Can edit and publish content for the venue.

Venue admin

Has all the editor's permissions plus security management; can register new editors and venue administrators.

Institution admin

Can manage users across all their venues; can register users in all their venues.
