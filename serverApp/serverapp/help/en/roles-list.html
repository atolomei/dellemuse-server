Help

Roles are used to grant permissions to users. A user can have more than one assigned role, and their permissions are all those granted by their roles. There are three types of roles, Generales, de Institution y de Site. Institution and site roles are created automatically with them.

  • General Roles
    • Admin
      This is the general administrator role, it has all available permissions, can see and do everything except edit the system user ('root')
    • Audit
      Can view all system content but cannot edit or modify anything
  • Institution Roles
    • Admin
      Can manage the institution and its sites. Examples:
      Institution Admin( MALBA )
      Institution Admin( Museo Nacional de Bellas Artes )
  • Site Roles
    • Admin
      Can manage and publish content and administer site users. Examples:
      Site Admin( MALBA )
      Site Admin( Museo Nacional de Bellas Artes )
    • Editor
      Can edit all site content but cannot manage security -edit users, change permissions, etc-, examples: Site Editor( MALBA ), Site Editor( Cabildo de Buenos Aires )
