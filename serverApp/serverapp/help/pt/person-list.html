Ajuda

Biblioteca de pessoas administrada pelos usuários com função geral Admin. Cada usuário do sistema, além de seus dados de usuário (senha, idioma, fuso horário, etc.) sempre tem uma pessoa associada onde constam seus dados pessoais (email, telefone, etc.).

Podem existir pessoas sem usuário associado, mas cada usuário sempre tem uma pessoa associada.
