Ajuda

As funções servem para conceder permissões aos usuários. Um usuário pode ter mais de uma função atribuída, e suas permissões são todas as que lhe concedem as funções que possui. Existem três tipos de funções, Generales, de Instituição y de Local. As funções de instituição e de local são criadas automaticamente com elas.

  • Funções gerais
    • Admin
      Esta é a função de administrador geral, possui todas as permissões disponíveis, pode ver e fazer tudo exceto editar o usuário do sistema ('root')
    • Audit
      Pode ver todos os conteúdos do sistema mas não pode editar nem modificar nada
  • Funções de Instituição
    • Admin
      Pode administrar a instituição e seus locais. Exemplos:
      Admin Instituição( MALBA )
      Admin Instituição( Museo Nacional de Bellas Artes )
  • Funções de Local
    • Admin
      Pode gerenciar e publicar os conteúdos e administrar os usuários do local. Exemplos:
      Admin de Local( MALBA )
      Admin de Local( Museo Nacional de Bellas Artes )
    • Editor
      Pode editar todos os conteúdos do local mas não pode gerenciar a segurança -editar usuários, alterar permissões, etc-, exemplos: Editor de Local( MALBA ), Editor de Local( Cabildo de Buenos Aires )
