Audio Studio

Console para gerar áudios de obras e introduções de audioguia.

1. Gerar o áudio de voz

O texto do áudio é retirado do conteúdo e a voz deve ser selecionada das vozes disponíveis no idioma ( biblioteca de vozes ).

2. Adicionar introdução de música

Este passo é opcional. A música pode ser selecionada dos arquivos disponíveis no sistema ( biblioteca de música ) ou colar uma URL do arquivo de música para que o sistema o baixe e utilize. Neste caso o usuário é responsável por ter a licença de uso do referido arquivo de música.

3. Integrar

Caso o áudio final seja integrado ao conteúdo, substituirá o áudio existente do conteúdo -se houver-, caso contrário ficará salvo no estúdio de áudio do conteúdo mas não será utilizado no audioguia.
